# 模板指南

使用 `drama-cut templates list` 查看内置模板。

- `default`：通用短剧投流素材。
- `conflict`：争吵、羞辱、复仇、权力博弈、当众打脸。
- `romance`：CP 关系、告白、误会、甜虐爱情。
- `suspense`：身份谜团、线索、秘密、合规悬疑。
- `family`：婚姻、亲子、婆媳矛盾、女性成长、家庭伦理。
- `content_clean`：优先做合规清理，同时保持故事可懂。
- `revenge`：压抑、揭露、反击、打脸节奏。
- `ancient_romance`：古装、仙侠、命运感、禁忌爱、情绪奇观。
- `cute_kids`：萌娃 hook、亲子情绪、身份揭露、认亲线。

如果用户给出特殊目标人群或平台，先选择最接近的内置模板，再把额外要求加入本次提示词或后续自定义模板。
